# Glossary

Workplace shorthand, acronyms, and internal language for the Jairo organization.

## Acronyms
| Term | Meaning | Context |
|------|---------|---------|
| AA | AutoAllies | Main project / product |
| ADO | Azure DevOps | Project management platform |
| SAFe | Scaled Agile Framework | Agile methodology used for auditing |
| PI | Program Increment | SAFe planning cadence |
| DoR | Definition of Ready | Description + Acceptance Criteria required |
| DoD | Definition of Done | Criteria for completing work items |
| BRD | Business Requirements Document | Strategic requirements doc |
| PRD | Product Requirements Document | Tactical product spec |
| UIUX | UI/UX Design | Design team focus area |
