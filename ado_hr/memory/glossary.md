# Glossary

Workplace shorthand, acronyms, and internal language for Jairosoft FINOPS HR auditing.

## Acronyms
| Term | Meaning | Context |
|------|---------|---------|
| APE | Annual Performance Evaluation | Employee yearly review process |
| PI | Program Increment | SAFe planning cadence (typically 8-12 weeks) |
| PI6 | Program Increment 6 | Current PI for 2026 |
| WIP | Work In Progress | Items actively being worked on |
| WFH | Work From Home | Employee work arrangement |
| BOD | Board of Directors | Jairosoft leadership |
| IC | Independent Contractor | Non-employee workers |
| CADAC | CADAC Training Seminar | Company training program |
| JIT | Jairosoft IT / Just-In-Time | Operations team or methodology |
| SAFe | Scaled Agile Framework | Framework used for auditing |
| ADO | Azure DevOps | Project management tool |

## Internal Terms
| Term | Meaning |
|------|---------|
| 201 file | Employee personnel file (Philippine HR term) |
| Iteration 6.4 | Current sprint: Feb 23 – Mar 8, 2026 |
| Stories and Deliverables | Board name for HR team backlog |
| Exit Clearance | Offboarding checklist for departing employees |
| Quitclaim | Legal document signed when employee departs |
| Probationary Contract | Employment contract during probation period |
